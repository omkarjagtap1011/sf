2. Create Custom Object

Go to:

Setup → Object Manager → Create → Custom Object

Create the object with:

Field	Value
Label	Student
Plural Label	Students
Object Name	Student
Record Name	Student Name
Data Type	Text

Click Save.

B. Class Field
